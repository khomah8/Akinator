## Installation
Use the package manager [yarn](https://yarnpkg.com/lang/en/) to install dependences.
```bash
yarn install 
```

## Create config file and paste your credentials
In project folder
```bash
cp etc/config.js.sample etc/config.js
```

## Starting
For starting project
```bash
yarn start
```
